# Audio Master Candidates

This note tracks candidate masters for the Itch.io web preview.

## Current Needs

- Title Bach cue with clean entry.
- Gradual Bach fade on first meaningful interaction.
- Ambient bed for La Soglia.
- Ambient bed for Il Giardino.
- Sparse evocative noises that can appear between commands.
- Revelation cue for the first Bach moment.

## Evaluation Criteria

- Starts reliably after browser user interaction.
- Does not feel like a second accidental music track.
- Loops cleanly in the browser.
- Leaves room for typewriter ticks and command feedback.
- Supports the suspended, oneiric tone without becoming decorative wallpaper.

## Release Notes

Before publishing, do one full browser listening pass from splash to preview ending. Mark any cue that feels too loud, too literal, or too repetitive for replacement.
